# Security Policy

## Supported versions

Security fixes are provided for the latest released major version of
`la-souris/document-signer-validsign`. Please ensure you are running a supported version before reporting
an issue.

## Reporting a vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public GitHub issue for security problems.**

Instead, email **ict@lasouris.nl** with:

- A description of the vulnerability and its potential impact.
- Steps to reproduce (proof of concept if possible).
- The affected version(s).

You will receive an acknowledgement as soon as possible. We ask that you give
us a reasonable amount of time to investigate and release a fix before any
public disclosure. We will keep you informed of our progress throughout.

Thank you for helping keep `la-souris/document-signer-validsign` and its users safe.
